C++11 Experiment Utilities

These are some useful utilities for running and maintaining experiments in c++11. This includes filesystem utilities, managers for reporting error and internal varriables, a manager for saving to files, and more. This is mostly useful for the managers to keep track of these variables for you. This is still being actively used and updated by me and the api is bound to change overtime. As it matures certain aspects will become stable.

auto-soft-wrap

A package for Atom. This enables soft wrapping on files based on their extension. There is also a customizable list of filetypes for which it considers.
